# Running Locally
## Frontend
- Go to app/frontend and run ```npm i``` for the first time
- and ```npm run dev```
## Backend
- Go to app/backend and run ```go mod download ```
- run ```air init``` to create the .toml file for the first time, or run ```air``` if it already exists
